# Pabasara Dewmini Portfolio

A responsive personal portfolio built with React, Vite, Tailwind CSS, and Framer Motion.

## Overview

This project showcases Pabasara Dewmini's profile as a Business Analyst and Software Engineering Undergraduate. It highlights skills, certifications, volunteering experience, projects, GitHub work, and contact details in a modern responsive layout.

## Features

- Responsive landing page for mobile, tablet, and desktop
- Animated UI using Framer Motion
- Skills, certifications, and volunteering sections
- Project showcase with modal details
- GitHub repository section
- Downloadable CV

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion

## Project Structure

```text
src/
  components/
  data/
  sections/
public/
  certifications/
  projects/
  cv.pdf
  profile.png
```

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start the development server

```bash
npm run dev
```

### 4. Build for production

```bash
npm run build
```

## Customization

- Update portfolio content in `src/data/`
- Replace images in `public/`
- Add certification images to `public/certifications/`
- Update project screenshots in `public/projects/`
- Replace the CV file in `public/cv.pdf`

This project can be deployed easily on:
